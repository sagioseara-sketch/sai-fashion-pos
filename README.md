# 👔 Sai Fashion POS

A professional Point of Sale (POS) billing system for clothing retail stores.

## 🌐 Live Website
**https://YOUR_USERNAME.github.io/sai-fashion-pos/**

---

## ✨ Features
- 🧾 Create bills with beautiful A4 PDF invoices
- 👥 Customer database with auto-fill
- 📦 Product inventory with low-stock alerts
- 📊 Sales dashboard with charts
- 📈 Daily/Weekly/Monthly reports + CSV export
- 📱 WhatsApp invoice sharing
- 🔒 Admin login system
- 💾 All data saved locally (localStorage)

## 🚀 Setup Instructions

### Run Locally
```bash
npm install
npm run dev
```

### Build for Production
```bash
npm run build
```

---

## 📱 GitHub Pages Deployment
Push to `main` branch → GitHub Actions auto-deploys to GitHub Pages.

**Setup once:**
1. Go to repo **Settings → Pages**
2. Set Source to **GitHub Actions**
3. Push any commit → site goes live!

---

## 📦 Android APK Build
1. Go to **Actions** tab in GitHub
2. Click **"Build Android APK"**
3. Click **"Run workflow"**
4. Wait ~15 minutes
5. Download APK from **Artifacts**
